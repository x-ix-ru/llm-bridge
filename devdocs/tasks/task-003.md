# Task 003 — Обновить и дополнить тесты OpenCode Config

## Описание

Обновить существующие тесты на OpenCode-конфигурацию с учётом новой формулы расчёта лимитов, а также добавить новые тест-кейсы для проверки конфигурируемости buffer и input.

## Покрытие требований

- Все существующие тесты `TestOpenCodeConfig_*` должны проходить с новыми значениями.
- Добавлены тесты на явный input (input > 0).
- Добавлены тесты на edge cases: малый context в auto mode, малый context + explicit input (guard-логика).

## Файлы для изменения

- `server/server_test.go` — обновление существующих тестов + новые тесты.
