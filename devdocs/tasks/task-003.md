# Task 003 — Обновить README.md Environment Variables

## Описание

Обновить секцию «Environment Variables» в `README.md`: добавить все 9 новых ENV-переменных к существующим `CONFIG_PATH` и `PORT`.

- Быстрый справочник в корне проекта.

## Покрытие требований

- Таблица `Environment Variables` должна содержать 11 строк (2 существующих + 9 новых).
- Формат таблицы: Variable | Description | Default.
